### groupby_task

### 설정 및 실행 방법

```
npm install mongodb
npm install monogoose
```
mongodb 연결을 위한 모듈 설치

```
npm install -g yarn
```
yarn 명령어 사용으 위한 모듈 설치

```
yarn add graphql
yarn add express-graphql
```
graphql 사용을 위한 모듈 설치

```
yarn add graphql apollo-server
```
아폴로 서버 사용을 위한 모듈 설치

```
node app.js
```
서버 실행
